## Recommended Learning Path

```
Week 1: LangChain Docs + LangChain Academy — foundations
Week 2: RAG Tutorial + Text Splitters — build your first pipeline
Week 3: LangGraph Docs + Academy Course — add agent capabilities
Week 4: RAGAS + LangSmith — evaluate and improve your system
Week 5: LangFuse + LiteLLM — production observability and cost
```

> **Tip:** Don't try to learn everything at once. Pick one level, build something small, evaluate it, then move to the next. The best way to learn is to **build -> measure -> improve**.
